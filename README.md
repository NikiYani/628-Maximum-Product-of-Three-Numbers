# 628. Maximum Product of Three Numbers

Given an integer array nums, find three numbers whose </br>
product is maximum and return the maximum product. </br>

## Example 1:

Input: nums = [1,2,3] </br>
Output: 6 </br>

## Example 2:

Input: nums = [1,2,3,4] </br>
Output: 24 </br>

## Example 3:

Input: nums = [-1,-2,-3] </br> 
Output: -6 </br>

## Constraints:

3 <= nums.length <= 104 </br>
-1000 <= nums[i] <= 1000 </br>
